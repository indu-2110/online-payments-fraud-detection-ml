## Project Description
This project aims to detect fraudulent online payment transactions using machine learning techniques. 
By analyzing historical transaction data and user behavior patterns, the system identifies suspicious 
transactions and helps prevent financial fraud in real time.

## Objectives
- Detect fraudulent transactions accurately
- Reduce false positives in fraud detection
- Improve online payment security using ML models

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib / Seaborn
- Jupyter Notebook
